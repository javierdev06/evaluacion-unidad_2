Acción: "Acceso no autorizado a cuentas":

Tipificación: Artículo 1 - Acceso ilícito. Se penaliza al que, sin derecho, acceda a un sistema informático superando barreras técnicas de seguridad.

Acción: "Distribución de malware (Wipers / Troyanos)":

Tipificación: Artículo 3 - Ataque a la integridad de los datos. El uso de malware diseñado para borrar datos (wipers) constituye la alteración, daño o destrucción ilícita de datos informáticos.

Acción: "Robar datos confidenciales":

Tipificación: Artículo 2 - Interceptación ilícita. Capturar o interceptar de manera técnica y sin autorización transmisiones no públicas de datos informáticos.

Acción: "Interrumpir operaciones de infraestructura crítica":

Tipificación: Artículo 4 - Ataque a la integridad de un sistema informático. Se configura al obstaculizar o interrumpir gravemente el funcionamiento de un sistema informático. Al ser contra el sector energético o de defensa, aplica la agravante del Artículo 6, por afectar servicios de utilidad pública o la seguridad nacional.
